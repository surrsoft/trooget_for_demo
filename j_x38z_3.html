[[w307w]] - использование тегов, [тег], [tag]
  • это технология использования тегов
  • принцип работы
    • подключить скрипт "_js/w307w/w307w.js"
    • в "блоки" которые нужно пометить тегами нужно поместить конструкцию "div class=w307w_tags". Внутри этой конструкции прописать теги в виде обычных span
    • остальное будет сделано автоматически - на базе имеющихся тегов будет добавлено "облако тегов" вверх страницы
    • при последовательном выборе тегов будут скрываться те "блоки" которые не содержат ВСЕХ выделенных тегов
    • по умолчанию "облако тегов" добавляется вверху страницы. Для размещения в любом месте страницы необходимо создать элемент "div id=w307w_tagSets"
